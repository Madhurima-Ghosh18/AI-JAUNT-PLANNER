# AI Jaunt Planner #

Designed and built an AI-powered trip planner using React, Google’s Gemini AI, Firebase, and Tailwind CSS. The app generates personalized itineraries based on budget and trip duration, leveraging AI for recommendations and Firebase for data management, with a responsive UI.

## Features ##

* AI-Powered Recommendations: Generates personalized travel itineraries based on user input (budget, trip duration, preferences) using Google’s Gemini AI.
  
* Real-time Data Management: Utilizes Firebase for user authentication, data storage, and real-time database updates.
  
* Responsive UI: Built with Tailwind CSS to ensure a responsive and user-friendly interface across various devices.
  
* Interactive Maps: Integration with map services for visualizing travel routes and destinations.
  
* User Authentication: Secure login and registration functionality with Firebase Authentication.

  
## Technologies Used ##

* React: JavaScript library for building user interfaces.
  
* Google’s Gemini AI: AI service for generating travel recommendations.
  
* Firebase: Platform for backend services including authentication, database, and hosting.
  
* Tailwind CSS: Utility-first CSS framework for styling.
  
* Vite: Frontend build tool for a fast development environment.
